# Public et chaos

## Ressource du public

Chaque membre du public gagne des **tomates** ou de l'**énergie de salle** au fil du temps. Les réactions coûteuses nécessitent davantage de ressource.

Ce système empêche le spam et crée des choix :

- gêner immédiatement ;
- économiser pour un événement majeur ;
- aider un comédien ;
- combiner une réaction avec les autres spectateurs.

## Réactions négatives

### Tomate

Ajoute une tache sur une zone limitée de l'interface du comédien.

Contre-jeux possibles :

- essuyer la tache ;
- déplacer le prompteur ;
- demander l'aide du souffleur ;
- continuer de mémoire ;
- obtenir des applaudissements.

### Huées

Réduisent temporairement la lisibilité sonore ou visuelle, mais ne doivent pas couper toute information essentielle.

### Téléphone dans la salle

Ajoute une sonnerie que les acteurs peuvent intégrer à la scène.

### Mauvais bruitage

Remplace ou superpose un son attendu.

### Changement d'émotion

Impose une nouvelle intention pendant quelques répliques.

### Mot interdit

Le comédien doit éviter un mot courant pendant une durée limitée.

## Réactions positives

- applaudissements ;
- rappel d'une réplique ;
- nettoyage partiel de l'écran ;
- temps supplémentaire ;
- indice ;
- boost de micro ;
- choix offert au comédien ;
- annulation collective d'un incident.

## Combinaisons

Certaines réactions deviennent plus fortes si plusieurs spectateurs les utilisent ensemble :

- pluie de tomates ;
- standing ovation ;
- rappel général ;
- changement de genre ;
- invasion de scène.

## Règle de lisibilité

Une perturbation doit respecter trois limites :

1. durée courte ;
2. zone ou information ciblée ;
3. possibilité de compensation.

Le système doit suivre un **budget de gêne**. Deux perturbations majeures ne peuvent pas recouvrir simultanément toutes les informations d'un même joueur.

## Public simulé

Le premier public automatisé doit être déterministe et fondé sur des règles.

Personnalités possibles :

- critique snob ;
- amateur d'explosions ;
- romantique ;
- impatient ;
- fan d'un acteur ;
- troll ;
- puriste du texte ;
- défenseur de l'improvisation.

Une IA générative pourra ensuite écrire les critiques, sans décider seule de la note.

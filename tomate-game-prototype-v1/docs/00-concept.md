# Concept général

## Pitch

**Tomate !** est un jeu de théâtre multijoueur asymétrique dans lequel les joueurs disposent de quelques secondes pour préparer une courte pièce, puis doivent l'interpréter et la mettre en scène sous la pression d'un public actif.

Selon le mode et le nombre de participants, les rôles peuvent être humains ou automatisés :

- comédien ;
- régisseur ;
- metteur en scène ;
- accessoiriste ;
- souffleur ;
- public ;
- narrateur ou maître de cérémonie.

## Promesse

Une représentation imparfaite mais mémorable vaut mieux qu'une exécution propre et sans surprise.

Le système ne cherche pas uniquement à mesurer la réussite. Il doit reconnaître :

- la récupération après une erreur ;
- l'improvisation cohérente ;
- le respect du thème ;
- la synchronisation entre les rôles ;
- la capacité à divertir le public ;
- le chaos produit sans détruire la partie.

## Positionnement

Le projet se situe à la rencontre de :

- party game ;
- jeu d'improvisation ;
- jeu de lecture et d'interprétation ;
- jeu de coordination asymétrique ;
- expérience de streaming participative ;
- création narrative assistée.

## Piliers

### 1. Pression temporelle

La préparation est volontairement courte. Les joueurs doivent prioriser et accepter que tout ne sera pas parfait.

### 2. Informations asymétriques

Chaque rôle voit une partie différente du spectacle. Personne ne détient seul toutes les informations utiles.

### 3. Réaction en chaîne

Les meilleurs incidents proviennent d'une erreur compréhensible qui pousse les autres joueurs à réagir.

### 4. Public actif

Le public vote, encourage, gêne, impose des contraintes et façonne le verdict.

### 5. Rejouabilité narrative

Les thèmes, personnages, objectifs secrets, incidents et réactions doivent se combiner pour produire des parties différentes.

## Anti-objectifs

Tomate ! ne doit pas devenir :

- un simple générateur de texte lu à voix haute ;
- une succession de malus aléatoires impossibles à contrer ;
- un jeu où le public élimine immédiatement le comédien ;
- une simulation réaliste de théâtre ;
- un produit dépendant d'un LLM pour fonctionner ;
- une expérience réservée aux joueurs à l'aise avec l'improvisation.

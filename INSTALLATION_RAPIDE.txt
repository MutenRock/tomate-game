TOMATE ! — PROTOTYPE MULTIJOUEUR V0.3
======================================

PREREQUIS
- Node.js 20 ou plus recent
- Tous les appareils sur le meme reseau Wi-Fi

WINDOWS
1. Clonez ou telechargez le depot.
2. Double-cliquez sur start_windows.bat.
3. Autorisez Node.js sur les reseaux prives si le pare-feu Windows le demande.
4. Ouvrez l'adresse "Local" sur le PC de l'hote.
5. Ouvrez l'adresse "Reseau" sur les telephones des autres joueurs.

CREER UNE PARTIE
- Choisissez 1 comedien pour jouer les deux roles.
- Choisissez 2 comediens pour attribuer un personnage a chacun.
- Les autres joueurs rejoignent comme public.
- En mode duo, le second joueur peut choisir "Comedien 2" en rejoignant.
- L'hote peut aussi promouvoir un membre du public depuis le salon.

LIRE LA SCENE
- DIDASCALIE : action a faire, ne pas lire a voix haute.
- INTENTION : emotion ou maniere de jouer.
- TEXTE A DIRE : seule zone a lire a voix haute.
- APRES LA REPLIQUE : action a faire avant de passer la parole.
- REPERE DE SCENE : information utile pour le rythme et le public.

LINUX
1. Ouvrez un terminal dans le dossier.
2. Lancez : ./start_linux.sh

TEST SUR UN SEUL PC
- Ouvrez deux ou trois onglets du navigateur.
- Creez une salle dans le premier.
- Rejoignez comme second comedien et/ou public dans les autres.

ARRET
- Fermez la fenetre du serveur ou utilisez Ctrl+C.

Aucune voix n'est enregistree. Utilisez Discord si les joueurs ne sont pas dans la meme piece.

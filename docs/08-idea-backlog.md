## Contraintes de comédien

- parler comme si tout était un secret ;
- ne jamais prononcer le nom d'un autre personnage ;
- transformer chaque question en accusation ;
- placer un mot absurde trois fois ;
- changer d'avis à chaque bruitage ;
- cacher qu'on est un fantôme ;
- convaincre le public que le décor est réel ;
- finir toutes les phrases de la même manière ;
- jouer quelqu'un qui ne comprend pas la scène ;
- défendre un objet sans importance.
